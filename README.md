👕 Elevate X – Online Clothing Store

Elevate X is a full-stack web application that allows customers to browse and purchase clothing online, while administrators can manage products, orders, and users. This project highlights full-stack development, RESTful APIs, database management, and frontend-backend integration.

👕 Features

Customer:

Register and log in
Browse products with images, descriptions, sizes, and prices
Add items to cart and place orders
View past orders and order details

Admin:

Dashboard displaying key stats (orders, products, users)
Manage products (add, update, delete)
Manage orders (approve, cancel, mark delivered)
Manage users and product categories

👕 Key Contributions
